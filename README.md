# 프로젝트
ATM 시스템

# 프로젝트 설명

이 프로젝트는 간단한 ATM 시스템을 구현한 것입니다.
사용자는 계좌를 생성하고, 잔액을 확인하고, 입출금을 할 수 있습니다.

# 기술 스택
- **Java 11+**: 프로젝트의 메인 프로그래밍 언어
- **Gradle**: 빌드 도구
- **JUnit 5**: 단위 테스트 프레임워크
- **Jacoco**: Jacoco Test Report 생성

# 사용 방법
1. ./gradlew build - 빌드 및 테스트 리포트 생성
2. Main.class 에서 Run 버튼을 눌려주세요
